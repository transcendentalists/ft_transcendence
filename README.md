# ft_transcendence

**ft_transcendence**는 루비 온 레일즈(RoR)과 BackBoneJS를 이용하여 실시간 게임 SPA를 만드는 프로젝트입니다.
1:1 실시간 게임 이외에도 게임 초대, 관전, 친구, 채팅, 길드, 길드전, 랭킹 등의 모듈이 모두 구현된 SPA입니다.
42서울의 1기 1차수 카뎃인 eunhkim, jujeong, yohlee, iwoo, sanam에 의해 수행되었습니다.
